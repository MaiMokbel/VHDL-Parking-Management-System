## Overview

This project implements a Smart Parking System using VHDL. The system simulates a parking management controller capable of monitoring vehicle entry and exit, tracking parking occupancy, and displaying system information on seven-segment displays.

Developed using Xilinx ISE for digital logic design and simulation.

---

## Features

- Parking slot occupancy management
- Vehicle entry detection
- Vehicle exit detection
- Vehicle counter
- Seven-segment display output
- Modular VHDL design
- Simulation-ready implementation

## How to Run

1. Open `SmartParkingSystem.xise` in Xilinx ISE.
2. Build the project.
3. Run the simulation.
4. Observe the waveform and verify parking system behavior.
